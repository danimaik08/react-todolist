# react-todolist

## Команды

1. Запуск на localhost: `npm start`
2. build `npm run build:dev` or `npm run build:prod`
3. jest `npm run test`

## BundleAnalyzer

add `USE_BUNDLE_ANALYZER = true` into your .env to use it
